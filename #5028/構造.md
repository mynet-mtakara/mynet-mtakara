## 実際の環境

```
@startuml
package 実行環境 {
package フレームワーク {
map コード {
}
}
}
@enduml
note left of 実行環境 : 本案や開発環境
```

## テスト環境

```
@startuml
package 実行環境 {
package ユニットテスト {
map コード {
}

}
}
@enduml
note left of 実行環境 : ローカルや札幌サーバ
note bottom of ユニットテスト : データや時間などテストしたい状況を作る
```
- コード側は実際に動く環境とコードを変えないようにする
- ユニットテスト側（テスト側）で実際に動いてる状況（テストしたい）を作る
